# ibm_project
IBM data analyst Capstone Project  Final Dashboard
The analysis was done on a dataset named Stack Overflow Developer Survey, which includes responses from developers worldwide, ​detailing the technologies they currently work with 
and those they wish to work with ​in the future. Various ​visualizations are created, that effectively display trends and relationships. Dashboards are created using IBM Cognos Analytics.
​The dashboards are divided into three distinct parts,each focusing on different aspects ​of technology usage and trends. ​These parts include Current Technology Usage Dashboard, 
Future Technology Trends Dashboard, ​Demographics Dashboard.
